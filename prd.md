# **PRD：趣味背单词 Web 应用（纯前端、本地存储版）**

## **1. 产品定位**

面向英语四级（CET-4）学生的趣味背单词网页应用。
目标是通过轻量游戏化、适度挑战、正向反馈，让用户愿意坚持每天背单词。
所有数据存储在浏览器本地，无需服务器和登录。

---

## **2. 核心目标（MVP）**

* 提供卡片式背单词学习流程
* 提供轻量测试（选择题）
* 实现每日任务（新词 + 复习词）
* 保持所有学习记录在本地（localStorage）
* 根据记忆状态自动安排复习
* 提供基本的进度展示

---

## **3. 核心功能模块**

### **3.1 单词学习（卡片模式）**

* 卡片正面：单词、音标、例句
* 卡片背面：释义、词性
* 操作：

  * 翻面
  * 标记状态：认识 / 模糊 / 不会
* 状态存储至 localStorage
* 用于后续测试与复习任务生成

---

### **3.2 趣味联想（可选项）**

* 每个单词展示一个有趣的记忆联想（本地文本或前端生成）
* 示例：abstract → “🤔 想不明白的东西就是抽象。”

---

### **3.3 测试模块（选择题）**

* 题型：

  * 英 → 中（默认）
  * 中 → 英（可选）
* 四选一
* 答题规则：

  * 答对 → 熟悉度 +1
  * 答错 → 状态下降级为 vague
  * 连续错 2 次 → 状态记为 unknown

---

### **3.4 每日任务系统**

每日任务包括：

* 新词（默认 20 个）
* 复习词（根据记忆状态自动生成）

间隔重复规则（SRS 简化版）：

* familiar → 3 天后复习
* vague → 明天复习
* unknown → 当天重复出现（最多 3 次）

任务生成写入 localStorage。

---

### **3.5 用户统计**

展示：

* 掌握词数
* 模糊词数
* 不会词数
* 累计学习天数
* 今日任务完成情况

图形方式：条形图或圆环图（纯前端绘制）

---

## **4. 页面结构（MVP）**

### **4.1 首页**

* 今日任务概览
* 按钮：

  * 开始学习
  * 开始测试
  * 查看进度

### **4.2 学习页（卡片）**

* 单词卡片（正反面）
* 状态按钮：认识 / 模糊 / 不会
* 进度条

### **4.3 测试页**

* 单选题
* 答案反馈（正确/错误）
* 下一题按钮

### **4.4 结果页**

* 今日任务完成提示
* 学习成果总结

### **4.5 进度页**

* 掌握度统计图
* 学习天数
* 历史记录（可选）

---

## **5. 数据结构（全部存储在浏览器本地）**

### **5.1 词库（写在前端代码中）**

```json
[
  {
    "id": 1,
    "word": "abstract",
    "phonetic": "/ˈæbstrækt/",
    "definition": "adj. 抽象的；理论的",
    "example": "This is an abstract concept."
  }
]
```

---

### **5.2 单词状态（localStorage: wordStatus）**

```json
{
  "abstract": {
    "status": "familiar",
    "lastReview": "2025-11-09",
    "reviewCount": 3
  }
}
```

---

### **5.3 每日任务（localStorage: dailyTask）**


---

### **5.4 用户统计（localStorage: stats）**

```json
{
  "totalLearned": 120,
  "learnDays": 7,
  "todayFinished": true
}
```

---

## **6. 用户流程图（文本版）**

```
进入首页
    ↓
判断是否为今日任务
    ↓（若不是）
生成新任务（新词 + 复习词）
    ↓
用户点击“开始学习”
    ↓
卡片学习流程（逐词学习）
    ↓
学习完成 → 进入测试
    ↓
选择题测试
    ↓
完成测试 → 任务完成
    ↓
写入统计数据
    ↓
展示结果页
```

---

## **7. MVP 范围（必须实现）**

* 卡片式背词
* 状态标记
* 本地存储（localStorage）
* 每日新词任务（20 个）
* 自动复习词生成（SRS 简化版）
* 选择题测试
* 基础统计页面
* 首页 + 学习 + 测试 + 结果页
